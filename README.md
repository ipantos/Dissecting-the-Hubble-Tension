## ğŸ“Š Methodology & Key Innovations

This project utilizes advanced statistical methods to analyze the consistency of Hubble constant measurements. Key analytical features include:

*   **Gaussian Kernel Density Estimation (KDE):** Construction of probability density functions (PDFs) for heterogeneous datasets.
*   **Statistical Weighting:** Incorporating individual measurement errors to accurately represent the statistical weight of each data point.
*   **Sensitivity & Bimodality Analysis:** A specialized analysis (via `h0_bimodality.py`) that identifies the impact of high-weight outliers. By removing specific measurements, we reveal a hidden bimodality in the "One-Step" distribution, suggesting distinct underlying populations in the data.
